# SafeWalk

SafeWalk is a safety-aware pedestrian navigation system that prioritizes personal security over distance or time.

## Architecture

- **Mobile**: Flutter application using Clean Architecture.
- **Backend**: Node.js/TypeScript with PostGIS for spatial data and routing.

## Goals

- **Safety-First Routing**: Routes based on risk minimization.
- **Privacy**, No passive tracking.
- **Community Trust**: Weighted community reporting.
